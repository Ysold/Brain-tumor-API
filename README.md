## Route

Take CSV file, train model, with randomForest:

```sh
"/training"
```

```sh
"/train_tensorflow"
```

Call model, if detect if tumor is detected:

```sh
"/predict"
```

Create a new model ...with many files :

```sh
"/training_classification"
```

Make call with Huggingface & Mistral library :

```sh
"/model"
```
